Anomalies Critiques Détectées

BLOQUANT

Instabilité du Serveur (HTTP 500)

Le serveur .NET crash systématiquement sur les endpoints POST, PUT et DELETE des modules Agency et Portal.
Ce défaut empêche toute création de véhicule ou de réservation.

MAJEUR

Défaut d'ID (HTTP 404)

Les requêtes de type GET /api/.../{id} renvoient une erreur 404, même pour des IDs existants. La logique de récupération individuelle est rompue.

Performance Cloud Azure

Certains appels (GET /Agency) affichent une latence de 2769ms. La performance est dégradée de 300% par rapport au seuil cible (800ms).

Verdict de l'Audit

"Le Backend est actuellement dans un état instable. Si la lecture des référentiels (Shared API) est fonctionnelle, la couche de persistance (écriture en base) est totalement inopérante. Une revue immédiate du code .NET (Exception Handling) est requise avant tout déploiement."
